## 快速开始

### 第一步：首次设置

```
/lover setup
```

系统会询问：
1. 想要男性还是女性恋人
2. 年龄范围偏好
3. 是否需要自定义名字

设置完成后，系统会询问是否授权数据收集（用于生成人格分析报告）。

### 第二步：回答问卷

```
/lover questionnaire
```

5 个问题，影响恋人的核心性格。也可以跳过直接用默认设置。

### 第三步：开始聊天

```
/lover talk 今天好累
```

## 导入个人数据（增强人格分析）

通过导入你自己的聊天记录和照片，可以让恋人更了解你的行为模式和审美偏好。

### 方式 A：微信/QQ 聊天记录

```
/lover import
```

系统会询问你聊天记录文件路径。支持的格式：`.txt`、`.html`、`.json`

**导出微信聊天记录方法**：
1. 打开微信 PC 版
2. 进入与好友的聊天窗口
3. 点击右上角 `...` → 导出聊天记录
4. 选择"包含图片/视频"或"仅文字"
5. 保存为 .txt 或 .html 格式

解析后可以提取：
- 沟通风格（简洁/详细）
- 口头禅和语气词
- 表情包使用习惯
- 活跃时段

### 方式 B：照片

```
/lover import
```

选择照片文件夹，系统会分析：
- 照片类型（风景/自拍/美食/人物）
- 拍摄地点分布
- 审美偏好和风格

### 方式 C：直接粘贴聊天内容

如果你不想导出文件，也可以直接粘贴聊天记录内容，我会分析你的沟通风格。

## 浏览器扩展（可选）

扩展可以收集你的浏览记录，帮助生成更准确的恋人画像。

### 安装步骤

#### 1. 打开 Chrome 扩展页面

```
chrome://extensions/
```

或：设置 → 更多工具 → 扩展程序

#### 2. 开启开发者模式

右上角开启 **开发者模式**

#### 3. 加载扩展

1. 点击 **加载已解压的扩展程序**
2. 选择 `~/.claude/skills/lover-skill/browser-extension/` 文件夹

#### 4. 固定到工具栏（可选）

点击 Chrome 工具栏的拼图图标 → 点击 Lover Skill 旁边的图钉图标

### 扩展功能

- 自动记录浏览的网页（URL、标题、时间）
- 追踪页面停留时长和阅读深度
- 每 10 分钟自动同步数据
- 数据保存在 `~/Downloads/lover-data/browsing.json`

> **注意**：首次使用扩展前，需要先在 Downloads 文件夹下创建 `lover-data` 子文件夹。Chrome 扩展无法自动创建目录，如果目录不存在会自动降级到 `~/Downloads/browsing.json`（备用路径）。

### 手动同步

点击扩展图标 → **同步数据**

### 验证安装

1. 点击扩展图标，打开扩展界面，确认浏览记录数在增加
2. 访问几个网页，等待几分钟后再次点击扩展图标查看记录数是否增加

### Edge 浏览器

同 Chrome，访问 `edge://extensions/` 操作。

## 恋人的 5 层人格结构

问卷答案和对话行为会驱动生成 5 层 Persona：

| 层次 | 内容 |
|------|------|
| Layer 0 | **硬规则** — 具体行为准则，不可违背 |
| Layer 1 | **身份** — 姓名、年龄、职业、背景 |
| Layer 2 | **说话方式** — 口头禅、emoji、对话示例 |
| Layer 3 | **感情逻辑** — 在乎什么、何时主动、如何表达爱 |
| Layer 4 | **边界** — 不喜欢的事、情绪触发器 |

Layer 0 是核心，决定了她不会是一个"无限配合你"的助手。

## 常见问题

**Q: 需要联网吗？**
A: 核心功能（对话、恋人生成）需要 Claude API或其他ai模型 连接。浏览器扩展的数据同步需要网络，但只会存在本地。

**Q: 数据存在哪里？**
A: 本地 `~/.claude/skills/lover-skill/data/`，加密存储。

**Q: 可以删除数据吗？**
A: 可以，运行 `/lover reset` 删除所有数据。

**Q: 不授权数据收集会怎样？**
A: 基础功能（对话、建议）正常使用，但无法生成详细的人格分析报告。

**Q: 扩展不显示图标？**
A: 正常，扩展没有内置图标。功能不受影响，只是工具栏显示默认拼图图标。

**Q: 如何让恋人记住更多？**
A: 多聊天，说具体的事情。每 6 轮对话会自动生成摘要保存。

**Q: 终端窗口关了怎么重新启动自动备份？**
A: 如果不小心关闭了自动备份终端，重新打开一个命令行窗口，运行：
```
cd %USERPROFILE%\.claude\skills\lover-skill && node scripts\auto-backup.js
```
保持窗口开着即可。也可以创建一个快捷方式方便下次一键启动。

**Q: 怎么确认自动备份在正常工作？**
A: 打开终端窗口，看到每 5 分钟打印一次 `[AutoBackup]` 开头的日志，说明正在运行。如果日志显示"备份完成！记录数: X"，说明数据正在同步。

**Q: 浏览器扩展的数据存在哪？**
A: 浏览器扩展数据先保存在 `~/Downloads/lover-data/browsing.json`，auto-backup 每 5 分钟同步到 `~/lover-data/browsing.json`。<br>**重要**：首次使用前需要在 Downloads 文件夹下创建 `lover-data` 子文件夹。如果扩展图标显示同步失败，说明目录不存在，扩展会自动降级到 `~/Downloads/browsing.json`。
